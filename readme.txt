Running
=======
To run this file, you must have both Python 2.4 (or higher)
(www.python.org) and pygame (www.pygame.org) installed.

Then to run carcode, unzip it to a working directory, and then run
trial1.py either at the command-line, e.g.

  $ python trial1.py

Or you can run python first and then import trial1.py and run the
main() function, e.g.

  $ python
  Python 2.4.2 (#1, Mar 20 2006, 17:38:54) 
  [GCC 4.0.0 (Apple Computer, Inc. build 5026)] on darwin
  Type "help", "copyright", "credits" or "license" for more information.
  >>> import trial1
  >>> trial1.main()


Commands for the Keyboard Car
=============================
                S - turn on/off engine
        up arrrow - accelerate
       down arrow - brake
left/right arrows - steer
                G - change gear to reverse/forward
              Z/C - turn left/right indicator lights
                H - honk horn
